torque_qhistory

qhistory is a basic script to make it easier to view PBS Torque accounting logs. It also includes a python module "torque_accounting" which encapsulates the actual parsing logic. The easiest way to understand it is to read the code (it's very simple).

usage: qhistory [-h] [--username USERNAME] [--queue QUEUE] [--job JOB]
                [--print-nodes] [--files FILES [FILES ...]]

optional arguments:
  -h, --help            show this help message and exit
  --username USERNAME, -u USERNAME
                        Show only jobs for this username
  --queue QUEUE, -q QUEUE
                        Show only jobs for this queue
  --job JOB, -j JOB     Show only this job ID in full
  --print-nodes, -n     Print full node list used for each job
  --files FILES [FILES ...], -f FILES [FILES ...]
                        Use the accounting records in these files

To build and install as an RPM

  1. python setup.py bdist_rpm
  2. sudo yum localinstall dist/torque-qhistory-0.1-1.noarch.rpm
